## User dashboard

*Redesigning Shipping Container Quotes With UX in Mind*



---

## The challenge

To create an intuitive and efficient module that empowers users to easily obtain accurate and competitive quotes for shipping containers to various accessible locations.

---

## Requirements and exploration

**Requirements:**
- Utilise the current collected prices and address data available in the database
- The shipping from locations will determine the available shipping to data available
- Prices will be shown for container sizes of 20ft and 40ft

**Exploration specifications:**
- Deviate from the design system only when needed
- Introduce interactive elements that emphasise the decision to click CTA
- The module will be placed in article grid size

![User dashboard overview](/media/projects/sirelo-interaction-design/overview.png)
---

## Current design analysis

The existing design presents shipping information in a table format within an article. It lacks sufficient incentive to encourage users to take action after reviewing the data in the table.

Through Hotjar recordings and page view analysis, we identified the need for an upgrade to improve the efficiency of obtaining shipping container quotes. Users consistently lost focus and deviated from the primary goal of requesting quotes — largely due to the absence of a clear, intuitive flow leading to the quote CTA.

---

## UX-driven insights

**1. User flow**
The current design lacks a clear and intuitive user flow towards the quote request process. Optimising the user journey requires establishing a logical and seamless path leading users to the quote CTA.

**2. Encouraging CTAs**
Providing compelling reasons for users to take action is crucial. Introducing persuasive elements within the design can prompt users to engage with the quote request module, converting their intent into a concrete inquiry.

**3. Streamlining the quote request**
User behaviour indicates the need for a more efficient quote request process. Implementing a user-friendly interface with interactive elements can simplify the process, reducing drop-offs and increasing engagement.

**4. Visual hierarchy**
Establishing a clear visual hierarchy is essential to draw users' attention effectively. Highlighting essential information and the CTA using visual cues and emphasis can guide users smoothly towards requesting a quote.

## Smaller notes — challenges during the project

### Optimising user choices — the invisible hand problem

One of the most significant challenges was a constraint posed by the current data collection and backend infrastructure. Users are required to select a starting location first, as the list of possible destinations is dependent on that initial choice.

To address this, we introduced a **shipping from toggle** with distinct states — prompting users to make an active, visible decision before moving forward. This approach ensures that the change in the destination list becomes more apparent, enabling a more seamless and informed user experience.

![Shipping from toggle — United Kingdom selected](/media/projects/sirelo-interaction-design/shipping-toggle.png)

### Optimising user choices — animations with purpose

Implementing small, purposeful animations tied to user inputs was a creative solution to enhance the experience of the interactive calculator. These animations were thoughtfully selected to add a sense of fun and freshness while keeping the primary goal — requesting a quote — front and centre.

By carefully balancing visual appeal with non-distracting behaviour, users' attention stays engaged without being pulled away from the essential components of the calculator.

## Outcome

By implementing these UX-driven enhancements — with a particular emphasis on the interactive calculator — we anticipated a substantial increase in user engagement and conversion rates.

The upgrade aimed to empower users, facilitating a swift and confident process of requesting shipping container quotes. The platform delivers a positive and rewarding experience, ensuring that users find the service reliable, efficient, and user-friendly.

By giving meticulous attention to user needs and preferences, this UX improvement elevates the platform's performance, cementing its position as a trusted and efficient solution for shipping container inquiries.

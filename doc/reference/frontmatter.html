📋 Frontmatter Reference

Complete reference for all frontmatter fields available in project files.

All Available Fields

Field Required Type Description
title Yes String Project title shown as page heading
client Yes String Company/client name. Used to link to experience cards.
date Yes String Project date in YYYY-MM-DD format
description Yes String Short summary for project cards (1-2 sentences)
tags No Array List of tags for filtering, e.g. ["UX", "UI"]
thumbnail No String URL for hero image on project cards
hidden No Boolean Set true to hide from grids

Example Frontmatter

---
title: "Efficiency and Fun | Designing a shipping calculator"
client: "Sirelo"
date: "2023-01-01"
description: "Designing a tool that's functional and — somehow — fun to use."
tags: ["Interaction Design", "UX", "Tool Design"]
thumbnail: "https://uploads-ssl.webflow.com/..."
---

Field Details

title

The main title of your project. This appears as:

  • Page heading on the project page
  • Title in browser tab
title: "My Project Title"

client

The company or client name. Important for linking to experience cards:

  • "Sirelo" → Shows Sirelo experience card
client: "Company Name"

date

Use ISO format YYYY-MM-DD. This determines sorting order.

date: "2024-01-01"

description

Short text shown on project cards in the work grid. Keep it concise (1-2 sentences).

description: "A brief description of the project."

tags

Optional array of tags. Currently displayed on project cards.

tags: ["UX", "UI Design", "Research"]

thumbnail

Hero image URL. Use Webflow CDN URL for best results.

thumbnail: "https://uploads-ssl.webflow.com/..."

hidden

Hide from grids but keep accessible via direct URL:

hidden: true  # Hidden from work grid
# or omit entirely  # Shown in work grid

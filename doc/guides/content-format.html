✍️ Content Formatting Guide

Learn how to format your case study content using markdown.

Basic Text Formatting

Bold Text
**Bold text here**

Renders as: Bold text here

Italic Text
*Italic text here*

Renders as: Italic text here

Bold + Italic
***Bold and italic***

Headings

Use headings to structure your case study:

# Main Title (H1)
## Section Title (H2)
### Subsection (H3)

Recommended Structure

## Overview
## The Challenge
## Research
## Design Process
## Outcome

Lists

Unordered List
- First item
- Second item
- Third item
Ordered List
1. First step
2. Second step
3. Third step
Nested List
- Parent item
  - Child item
  - Another child
- Another parent

Images

![Alt text describing image](image-url.png)

Image Best Practices

  • Always include descriptive alt text
  • Use consistent naming for images
  • Optimize images before uploading

Links

External Link
[View on Behance](https://behance.net/project)

Section Dividers

Use horizontal rules to separate sections:

---

## Next Section

Blockquotes

> This is a quote or callout

Code Blocks

`inline code`

```
code block
multiple lines
```
